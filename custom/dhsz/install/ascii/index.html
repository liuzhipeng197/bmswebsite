您需要FTP软件上传NITC系统，如果您没有FTP软件，请点击 FTP软件下载 并安装好。
下面将简单介绍FTP软件安装及用法
解压步骤就省略了，直接打开解压好的FlashFXP文件夹：

依次点击：“站点”菜单--“站点管理器”--新建站点--添加您的空间地址、帐号及密码--应用--关闭站点管理器

站点添加完之后接下来是设置FTP为二进制上传方式：
依次点击：选项菜单--参数设置--传送--传送模式--选择“二进制(图像)”--确定--关闭FlashFXP软件后重新打开

FLashFXP二进制上传方式设置完后，接下来准备上传网站：
从左侧本地目录找出下载的NITC系统，进入文件，选中所有文件单击右键传输至您的空间


注：有时候您空间的根目录可能会在wwwroot下面，所以上传的时候注意要先进入服务器空间的根目录。
